PNG

PNG stands for Portable Network Graphics. It is good for when you want to edit an image (ex. a transparent bcakground). It will not let you loose quality of photos. Best times to use PNG: - Small images - Images with transparent backgrounds

JPEG

JPEG stands for Joint Potographic experts Group. You can change the quality of the photo (low, medium or high). Takes up less space, but the quality can be comprimised when editing. Best Times to use JPEG: - Colorful images - If you want to reduce file size

GIF

GIF stands for Graphics Interchange Format. You can create animations using this format. Less color range. Best Times to use GIF: - Simple Images - Animated Images
